# Study Progress Tracker

Ein einfaches Studien-Dashboard als Kommandozeilenprogramm in Python.

## Voraussetzungen

- Python 3

## Start

```bash
python3 main.py
```

## Tests

```bash
python3 tests.py
```

## Funktionen

- Dashboard anzeigen
- Module hinzufuegen und loeschen
- Noten eintragen
- offene und bestandene Module anzeigen
- aktuelles Semester festlegen
- Zieldatum aendern
- Daten in `data.json` speichern und laden
